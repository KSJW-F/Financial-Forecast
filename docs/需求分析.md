# AI 金融报告分析市场走势 — 需求分析文档

## 1. 项目背景

期货市场分析师会发布日评、早评等研报，内容包含对具体品种后续走势的判断。本项目目标是对已采集的研报进行自动提取、清洗、趋势分析，并将结果入库展示，辅助用户观察某品种在一段时间内的观点变化。

## 2. 功能需求

| 编号 | 需求 | 说明 |
|------|------|------|
| F1 | 数据读取 | 从数据库读取文章元数据；首次导入时从 `data/` 目录扫描 HTML/PDF/PNG |
| F2 | 内容提取 | 支持 HTML、PDF、PNG（OCR 可选）格式正文提取 |
| F3 | 内容清洗 | 去除免责声明、导航/footer 噪声，统一空白字符 |
| F4 | 规则提取 | 对含“操作建议”“偏多/偏空”等明确表述直接提取趋势 |
| F5 | LLM 分析 | 无明确表述时调用大语言模型判断趋势 |
| F6 | 结果入库 | 保存文章与预测结果 |
| F7 | 列表展示 | 展示期货公司、标题、日期、观点摘要 |
| F8 | 详情展示 | 点击文章查看正文与解析结果 |
| F9 | 图表展示 | 按品种展示趋势评分走势与分布 |

## 3. 非功能需求

- 语言：Python 3.11+
- 数据库：MySQL（生产）/ SQLite（本地开发默认）
- 可配置 LLM API Key
- 支持批量导入与分页浏览

## 4. 技术选型

| 模块 | 技术 | 理由 |
|------|------|------|
| Web 框架 | Flask | 轻量、易部署，满足列表/详情/图表页面 |
| ORM | SQLAlchemy | 支持 MySQL/SQLite 切换 |
| HTML 解析 | BeautifulSoup + lxml | 适配多期货公司页面结构 |
| PDF 解析 | pdfplumber | 纯 Python，部署简单 |
| 可视化 | Plotly | 交互式趋势图 |
| LLM | OpenAI 兼容 API | 可选启用，规则分析为默认 |

## 5. 数据说明

- 数据目录：`data/YYYYMMDD/机构名_文章ID_序号.{html|pdf}`
- 样本量：约 7000+ 文件
- 主要格式：HTML、少量 PDF

4. 明确观点文章可通过规则直接解析
5. 文档齐全（需求、设计、README）
